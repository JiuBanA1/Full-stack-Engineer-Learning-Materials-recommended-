## 四、HTML基本结构
```html
<html>
  <!--网页头部-->
  <head>
    <title>我的第一个网页</title>
  </head>
  <!--主体部分-->
  <body>
    我的第一个网页
  </body>
</html>
```
## 五、DOCTYPE声明
```html
<!DOCTYPE html>
<!--html告诉浏览器使用什么规范-->
<html>
<head lang="en">
    <meta charset="utf-8"/>
    <title>新建标签页</title>
</head>
<body>

</body>
</html>
```

## 十六、锚链接
> 从A页面的甲位置跳转到本页中的乙位置
> 从A页面的甲位置跳转到B页面中的乙位置

### 1、创建跳转标记
```html
<a name="marker">乙位置</a>
```
### 2、创建跳转链接
```html
<a href="#marker">甲位置</a>
```

## 十八、行内元素和块元素
### 块元素
> 无论内容多少，该元素独占一行（p、h1-h6…）

### 行内元素
> 内容撑开宽度，左右都是行内元素的可以排在一行(a、strong、em…)
